##Описание проекта
####Проект представляет собой приложение, в котором студент может решать задачи и проверять правильность их решения
####Для каждого пользователя (студента) предусматривается свой вариант, результаты в дальнейшем записываются в БД
## Реализация
- Студент видит интерфейс, в котором преполагается ввести значение в поля 
####`введите фамилию` 
####`введите имя` 
####`выберете группу` 

- Далее переходим к `Вариант №...` где выводится список задач

- Студент видит список и начинает решать предложенные вариантом задачи

- После того, как задача решена, нужно вписать результат в поле `введите ответ`

- Система проверяет правильность решения и дает понять, верно решена задача или нет
####`верный ответ`
####`неверный ответ`

- После того, как студент завершает свой вариант, он нажимает `завершить` и система считает количество верно и не верно решенных задач
